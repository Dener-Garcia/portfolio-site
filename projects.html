Projeto: Web site Company Geradores

Energia para momentos díficeis

Cliente

Company Geradores

Entregas

UI Interface no Figma, web site responsivo

A Company geradores é uma empresa que presta serviços de instalação e manutenção em grupos geradores a gasolina ou diesel.

Objetivo do projeto

Criar a presença digital da Company Geradores através de um web site que seja totalmente responsivo, no projeto foi utilizado um layout base desenvolvido no Figma por mim e posteriormente realizado a codificação em HTML, CSS e Javascript seguindo boas práticas de design e código.

Processos de UX

Discovery, sitemap e wireframe

Mesmo sendo um projeto relativamente simples, algumas escolhas foram feitas cuidadosamente, escolhi um site one page pois a maior parte dos usuários fariam a visita usando um smartphone dessa forma privilegiamos o scroll da página. O site teria poucas opções para o usuário não se perder, considerando o meio que o cliente trabalha ter velocidade na resposta do atendimento era crucial. Por fim o usuário tem sempre disponível um botão de contato direto com o prestador de serviço, tornando o contato simples e efetivo.

Style Guide do projeto

Cores e tipografia

Uma das exigências do projeto era a utilização das cores azul e laranja, essas foram adaptadas para a UI pois as cores antigas eram baseadas em um cenário gráfico. Trabalhei com as proporções de 60 30 10 para distribuir de forma harmônica as cores pela interface. A fonte utilizada foi a Roboto, uma fonte pensada para a web já que possui vários estilos se adaptando as várias necessidades de uma interface.

Componentização

Agilidade e escalabilidade

Desenvolvimento baseado em componentes permite manter uma coerência e padrão de design, criando componentes no figma diminuimos sua complexibilidade na hora do desenvolver "codar" pois ele pode entender como funciona cada recurso e como ele deve se comportar.

Workflow

Design centrado no usuário

Uma interface com estrutura de 5 elementos, o processo de criação foi focado na estratégia, escopo, estrutura, esqueleto e por fim superfície. Procurei entender os sites dos concorrentes principalmente as funções que eram comum a todos eles, adaptando essas necessidades a uma interface própria que seja focada no usuário e que traga agilidade para o usuário encontrar o que precisa e entrar em contato com o profissional de forma rápida, além de conhecer as habilidades e serviços do mesmo. Um dos pontos críticos do projeto era ter uma boa pontuação de CEO para que o site seja rankeado no google trazendo mais leads para a empressa.

Dificuldades

Desafios durante o projeto

Criar um design diferente mas sem sair do nicho da empresa foi um dos desafios em que tive de me concentrar, para quem procura esse tipo de serviço é necessário passar algumas sensações ao usuário final como confiança e profissionalismo então cada elemento foi pensado para ter um conceito e uma função especifica.

Projeto: Calculadora de resisão trabalhista

Uma ferramenta para auxiliar as pessoas

Cliente

Colaboração DEV

Entregas

UI Interface no Figma com handoff para time de desenvolvedores

Colaboração DEV é uma iniciativa de juntar vários desenvolvedores para criar aplicações e com isso aprender fazendo

Objetivo do projeto

Desenvolver uma calculadora para resisão trabalhista que seja funcional e responsiva para ser utilizada em qualquer dispositivo. Realizei todo o layout no Figma pensando principalmente na facilidade de utilizar a ferramenta, foquei em tirar a complexibilidade de termos ou jargões da área de contábeis.

Style Guide do projeto

Cores, tipografia e design tokens

Pensando numa aplicação moderna utilzei o conceito de dark theme afim de colaborar para a acessibilidade e contraste das cores com seus componentes. A fonte escolhida foi a Inter por ser cuidadosamente trabalhada e projetada para telas.

Componentização

Agilidade e escalabilidade

Desenvolvimento baseado em componentes permite manter uma coerência e padrão de design, criando componentes no figma diminuimos sua complexibilidade na hora do desenvolver "codar" pois ele pode entender como funciona cada recurso e como ele deve se comportar.

Workflow

Imersão, ideação, prototipagem e implementação

Para construir a aplicação realizei um desk research de ferramentas similares para entender a usabilidade e os recursos. Após todo layout criado no Figma todo nosso time do Colaboração DEV é divido em squads e utilizamos o Scrum para dividir e monitorar as tarefas a serem realizadas, por fim passamos para os testes isolados e finalmente fazemos o deploy da aplicação.

Dificuldades

Desafios durante o projeto

Mesmo sendo um projeto fictício, temos prazos bem definidos para tudo, somos em torno de 40 devs e somente eu como designer do grupo, tive dois dias para criar todo o layout da interface e temos 2 semanas para realizar a entrega do projeto como um todo. O PO product ownwer fica responsável por acompanhar o andamento do projeto, e nós também fazemos a gestão de tudo pelo Trello.
